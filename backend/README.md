# 📝 To-do-list BE

> Restful API로 구현한 백엔드서버입니다.

<br>

## ▶ Run

```
    $ git clone https://github.com/DDD05/To-do-list-noti2007.git
    backend 디렉토리로 이동
    run TodoappApplication.class
```

## 🔨 사용 기술

 - Java
 - Spring Data JPA
 - MySQL
